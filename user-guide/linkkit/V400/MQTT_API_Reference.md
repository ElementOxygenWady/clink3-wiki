## <a name="aiot_mqtt_init">aiot_mqtt_init</a>
原型
---
```c
void *  aiot_mqtt_init(void);
```

接口说明
---
创建一个MQTT的本地对象, 该对象经创建和配置后, 可用于建立MQTT通信连接

参数列表
---
无

返回值
---
| 值      | 说明
|---------|-------------
| NULL    | 创建失败
| 非NULL  | MQTT句柄

参数附加说明
---
无
